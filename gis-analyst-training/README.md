# GIS Analyst Training

Source materials for the [GIS Analyst Training Workshop](https://firmanhadi21.github.io/blog/materials/20260326_GIS_Analysis_Workshop.html).

## Contents

R Markdown source files for the training bookdown:

- `index.Rmd` - Course overview
- `01-intro.Rmd` - Introduction to GIS
- `02-Day-2.Rmd` through `09-Day-9.Rmd` - Daily modules covering PostGIS, spatial SQL, QGIS, and web GIS

## External Data Sources

The course uses data from:
- [Geofabrik](https://download.geofabrik.de/) - OpenStreetMap extracts (roads, buildings, POIs)
- [Kemendagri ArcGIS Service](https://gis.dukcapil.kemendagri.go.id/) - Indonesian administrative boundaries
- [WorldPop](https://www.worldpop.org/) - Population density rasters

## Software Required

- PostgreSQL + PostGIS
- QGIS 3.x
- GeoServer (for web GIS modules)
