# Irrigation Performance Analysis

Code for the [Understanding Irrigation Performance Analysis](https://firmanhadi21.github.io/blog/materials/20260328_Irrigation_Performance_Tutorial.html) course.

## Contents

- `PUPR_IrrigationPerformance_5Oct25.ipynb` - Jupyter notebook for irrigation performance evaluation using Google Earth Engine, MODIS ETa, ERA5-Land, and Sentinel-2/Landsat imagery

## Requirements

- Python 3
- Google Earth Engine (GEE) account and service account credentials
- pandas, earthengine-api

## Study Area

DI Klambu irrigation district, Central Java, Indonesia (749 tertiary blocks).
